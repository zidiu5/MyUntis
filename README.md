# MyUntis

Eine inoffizielle Android-App für WebUntis, entwickelt mit Jetpack Compose.

## Features
- Stundenplan (Tag / Woche / Monat) mit Offline-Cache
- Hausaufgaben mit Checkbox-Funktion
- Noten & Fehlstunden
- Einstellungen: Dark Mode, Fachfarben
